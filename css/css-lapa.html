Kas ir CSS?

Cascading Style Sheets ir stila lapas valoda, ko izmanto, lai aprakstītu dokumenta noformējumu, kas rakstīts iezīmēšanas valodā, piemēram, HTML vai XML.

CSS ir globālā tīmekļa stūrakmens tehnoloģija līdzās HTML un JavaScript.
